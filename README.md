Este projeto consiste em uma simulação educacional de uma página web clonada de uma aplicação popular, desenvolvida com o objetivo de estudar, em ambiente controlado, os mecanismos técnicos utilizados em ataques de phishing e os fatores que tornam esse tipo de golpe eficiente.

A aplicação foi criada para fins exclusivamente acadêmicos e laboratoriais, permitindo a análise de:

clonagem visual de interfaces web;

captura de credenciais em ambiente simulado;

armazenamento de dados em banco de dados;

hospedagem em servidor Apache utilizando Ubuntu Server com pacote LAMP;

transferência de arquivos via FTP;

configuração de DNS local para simular um domínio funcional.

O projeto foi executado em máquinas virtuais, com foco no aprendizado prático sobre segurança da informação, administração de servidores e conscientização sobre ameaças digitais.

Aviso: Este projeto foi desenvolvido exclusivamente para fins educacionais, em ambiente isolado e controlado, sem qualquer finalidade maliciosa.
